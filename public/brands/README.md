# Brand logo assets

The public landing page (`/`) renders these official brand logos in the
**Brands** section. If a file is ever missing, that card falls back to a coloured
dot + brand name — no broken images.

## Current files

| File           | Brand                | Notes                                             |
| -------------- | -------------------- | ------------------------------------------------- |
| `bookends.png` | Bookends Hospitality | White wordmark on the blue brand plate            |
| `capiche.png`  | Capiche              | Black "Capiche" script wordmark on white          |
| `aiko.png`     | Aiko                 | Black "AIKO" wordmark on cream                     |

These were generated by tightly cropping the official logo artboards from the
Illustrator source files in `/assets/*.ai` (each `.ai` is a multi-page brand
guideline; the logo sits centred on one artboard). Each PNG is the brand's
native logo lockup, shown as a rounded plate ~48px tall.

## Replacing with vector (recommended for print-grade crispness)

Export an **SVG** from the `.ai` in Illustrator (File → Export As → SVG) and save
it as `bookends.svg` / `capiche.svg` / `aiko.svg` here, then delete the matching
`.png`. The page loads `{brand}.png` first, then `{brand}.svg`.
